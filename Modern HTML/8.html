What is HTML?

HTML (HyperText Markup Language) is the standard markup language for creating web pages and web applications. It provides the structure and content of a webpage, defining the elements and their relationships.

What is Semantic HTML?

Semantic HTML refers to the use of specific HTML tags that carry meaning and provide a logical structure to web content. By using semantic tags like <header>, <nav>, <main>, and <section>, we can convey the purpose and relationships of different parts of a webpage.

Commonly Used Semantic Tags

  • <header>
  • <nav>
  • <main>
  • <section>
  • <article>
  • <footer>
