What is web accessibility? List some of the assistive devices which play a major role in providing accessibility.

Web accessibility refers to the practice of designing and developing websites and web applications that can be accessed and
 used by all individuals, including those with disabilities. It aims to ensure that people with disabilities can perceive,
  understand, navigate, and interact with web content effectively.

Assistive devices are tools or technologies that assist individuals with disabilities in accessing and using digital content. 
Here are some major assistive devices that play a significant role in providing web accessibility:

1. Screen Readers
2. Screen Magnifiers
3. Braille Displays
4. Speech Recognition Software
5. Keyboard Accessibility
